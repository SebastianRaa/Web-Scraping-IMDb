# Web Scraping IMDb

This python script scrapes the IMDb search page for movies.

Currently, 25 movies with will be shown. They have to have at least 1,000 reviews. The are sorted by populartity. Filters can be used, but they are optional. For each movie, the following data will be displayed: Title, release year, runtime, pg-rating, IMDb rating, number of reviews and a description.

## Filters

- time period
- min. rating
- genre
- max. runtime
